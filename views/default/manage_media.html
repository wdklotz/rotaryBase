[1] You can use the Add Record action to upload a media file (for example an image) and link it to a page. Mind that media files are linked to pages and inherit access permission from the pages.

This Media page shows all media files uploaded and lists the proper page a file is linked to.

If, for example, you upload an image named test.jpg with title beach, the link expression will be something like: @////15/beach.jpg. The id of the record storing the media file is 15, beach is its title, .jpg is the extension of the original file. When clicking copy, this link expression is automatically pasted into the bottom of the wiki page. Because it is pasted as html comment the page view does not change.

To activate and position the image in the wiki page you have to edit the page source.
